Array 매서드의 indexOf() 사용

var furit = new Array("apple","banana","orange","melon");
document.write(furit.indexOf("apple"));
+add tip

1. 값이 없으면 -1을 반환한다.
2. lastIndexOf()는 마지막 검색한 값을 반환합니다.
3. 시작 인덱스를 매개변수로 사용해 검색이 시작하는 지점을 설정할 수 있다.
var furit = new Array("apple", "banana", "orange", "melon","apple"); document.write(furit.indexOf("apple")); document.write(furit.lastIndexOf("apple"));

find()

5. 두 메서드 모두 원래 배열을 바꾸지는 않습니다.

concat()과 apply()를 사용해 2차원 배열을 1차원 배열로 만들기
+tip
Array 객체의 concat() 메서드를 사용해 다차원 배열을 1차원 배열로 평평하게 만들 수 있다.

var num = [];
num[0] = [1,2];
num[1] = [3,4];
num[2] = [5,6];
num[3] = [7,8];
// 평평해진 배열
var newArray = num.concat.apply([],num);
document.write(newArray[5]);
+add tip

1. concat()메서드는 한 개 이상의 배열을 매개 변수로 받아서, 메서드를 호출한 부모 배열의 끝에 변수로 받은 배열 요소를 추가함.
var newArray = num[0].concat(num[1],num[2],num[3]);
2. 장황하고 오류나기 쉬움. (루프나 재귀도 마찬가지)
3. apply() 메서드는 한수에 주어진 배열 인수들로 concat() 메서드 호출
배열 오소 삭제 또는 치환
+tip
Array 객체의 index()와 splice()를 사용해 배열 요소를 삭제/치환.
var fruits = new Array("apple","banana","pineApple","orange");
// 배열에서 요소 삭제하기
fruits.splice(fruits.indexOf("apple"),1);
// 새로운 요소 삽입하기
fruits.splice(fruits.lastIndexOf("banana"),1,"berry");
+add tip

1. splice()메서드는 세개의 매개변수를 가짐.
2. 첫번째 매개변수는 삽입/제거하려는 위치의 인덱스로 반드시 입력 / 두번째 매개변수(선택적)는 삭제하려는 요소의 개수 / 세번째 매개변수(선택적)는 치환할 요소들.
3. 인덱스가 음수이면 요소는 배열의 시작 부분이 아닌 배열의 끝 부분에서부터 삭제됨.
삭제하려는 요소의 수가 주어지지 않으면, 인덱스에서 시작해 배열 끝까지 모든 요소가 삭제 됨.
var fruits = new Array("apple", "banana", "pineApple", "orange", "banana");
// 새 요소 삽입
fruit.splice(-1 , 1 , "monkey");
ar fruits = new Array("apple", "banana", "pineApple", "orange","Kiwi");
// 두 번째 배열 끝까지 모든 요소 삭제
fruit.splice(2);
var fruit = ["apple", "banana", "pineApple", "orange","Kiwi"];
